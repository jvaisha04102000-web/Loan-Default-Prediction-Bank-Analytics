# Bank Loan Default Prediction – Machine Learning Project

## Problem Statement
Banks face risk when customers fail to repay loans. This project predicts the probability of loan default using machine learning.

## Project Workflow
1. Dataset Generation using Python
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training (Logistic Regression)
6. Model Evaluation
7. Streamlit Prediction Dashboard

## Technologies Used
Python
Pandas
NumPy
Scikit-learn
Streamlit

## Project Structure
Bank_Dataset.py
EDA_bank_project.py
model_training.py
loan_prediction_app.py
bank_loan_dataset.csv

## How to Run the Project

```bash
pip install -r requirements.txt
streamlit run loan_prediction_app.py
```
